InfinityNet API

API REST moderna para gerenciamento de usuários e autenticação

Informações da API v1.0.0 Online

Esta API REST foi construída com Node.js e TypeScript, seguindo as melhores práticas de arquitetura, segurança e escalabilidade. Ela oferece um conjunto completo de endpoints para gerenciamento de usuários, perfis e autenticação.

Verificar Status

Endpoints Disponíveis

Autenticação

/api/auth

Endpoints para login, refresh token e gerenciamento de credenciais.

Usuários

/api/users

Gerenciamento completo de usuários com suporte a CRUD.

Perfis

/api/roles

Gerenciamento de perfis e níveis de acesso ao sistema.

Status

/api/status

Gerenciamento de estados para usuários (ativo, inativo, etc.).

Verificação de Saúde

/api/health

Endpoint para verificar o status e funcionamento da API.

Segurança

Esta API implementa medidas de segurança robustas, incluindo:

  • Autenticação JWT com refresh tokens
  • Controle de acesso baseado em perfis (RBAC)
  • Proteção contra ataques comuns (XSS, CSRF, etc.)
  • Limitação de taxa para prevenir abusos
  • Validação rigorosa de dados com Zod
  • Criptografia de senhas com bcrypt
